## Quickstart

### 1. Installation
```bash
git clone https://github.com/hohopilya-ctrl/neon_gridiron.git
cd neon_gridiron
pip install -e .[dev]
```

### 2. Verify Install
```bash
neon-gridiron check
```

### 3. Launch the Stack
```bash
neon-gridiron ultra
```

## Features
- **Deterministic AI**: Bit-identical simulation reproducibility.
- **Gymnasium Core**: Standard Gymnasium environment for RL.
- **League Engine**: Real Elo-based competitive evaluation.
- **Futuristic Mechanics**: Sonic blasts, phase shifts, and neon dashing.
- **Live Telemetry**: V2 high-performance protocol for 3D visualizers.

## CLI Usage
- `neon-gridiron train`: Start RL league training.
- `neon-gridiron server`: Launch the WebSocket telemetry bridge.
- `neon-gridiron replay <file>`: Compare or playback simulation runs.
